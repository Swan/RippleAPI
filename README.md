# Usage
It's extremely simple to start using RippleAPI.js just by following the steps below.
* Run `npm install rippleapi`

```js
const rippleAPI = require('rippleapi');

rippleAPI.ping()
    .then((response) => {
        console.log(response);
    })
    .catch((err) => {
      throw new Error(err);
    });
```
